# Magnetorquer Test PCB

## Overview
This PCB is designed to test different magnetorquer coils under varying operating conditions like voltage, current, and temperature.

The board uses an INA219 for current sensing and ADC voltage dividers for voltage measurement. 
It also has I²C and CAN headers so external sensors like magnetometers and temperature sensors can be connected if needed.

## Features
- DRV8871 motor driver
- INA219 current sensing
- ADC voltage measurement using voltage dividers
- USB Type-C interface
- SWD programming/debugging header
- I²C expansion header
- CAN header

## Applications
- Testing different magnetorquer coils
- Characterizing coil performance under different conditions
- Connecting external sensors for additional measurements
- Closed loop testing with magnetometers
